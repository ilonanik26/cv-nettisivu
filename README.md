# Ilona Nikulina | Portfolio & CV

A bilingual portfolio and CV website built to present my skills, projects, and background in a more visual and personal way than a traditional CV.

## About The Project

This website combines my portfolio and CV into one place. It showcases my interests in frontend development, UI/UX design, and data analytics, while also giving a clearer picture of who I am, what I have built, and what kind of opportunities I am looking for.

The site includes both Finnish and English versions, so it works better for both local and international applications.

## Features

- Bilingual interface: Finnish / English
- Responsive design for desktop and mobile
- Aurora-inspired visual theme
- Interactive skills section with expandable categories
- Portfolio section with selected projects
- Contact section with modal popup for email and phone
- Copy-to-clipboard feature for contact details
- Continuous tool carousel

## Built With

- HTML
- CSS
- JavaScript

## Projects Included

- Budget App
- Game Website
- Poshlaya Molly Website

## Purpose

I created this website to support internship, summer job, and junior-level applications.
It is meant to work as both a personal portfolio and an extended CV.
